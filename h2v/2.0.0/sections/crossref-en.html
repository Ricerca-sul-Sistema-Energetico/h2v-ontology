Hydrogen Storagec back to ToC or Class ToC

IRI: https://w3id.org/h2v#HydrogenStorage

A Hydrogen Storage is the system for storing the produced hydrogen, which can be stored in compressed gaseous form, as a liquid, or through other methods. Hydrogen storage is essential to ensure system flexibility and to balance energy production and demand. The Hydrogen Storage is the interface for import/export operations. The Hydrogen Storage is part of the macrocomponent Hydrogen Storage Area.

Hydrogen Valleyc back to ToC or Class ToC

IRI: https://w3id.org/h2v#HydrogenValley

A Hydrogen Valley (HV) is a geographic area—such as a city, region, island, or industrial cluster—where various hydrogen applications are combined into an integrated hydrogen ecosystem that consumes a significant amount of hydrogen. An Hydrogen Valley can therefore encompass the entire hydrogen value chain: production, storage, distribution, and end use.
has super-classes
System c
